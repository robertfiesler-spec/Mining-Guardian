# Slack Branding — 90-Second Checklist

You need to do these 4 manual uploads in the Slack UI. **All 4 use the same file:**

`/Users/BigBobby/Documents/GitHub/Mining Gaurdian/branding/mining_guardian_mg_icon.png`

Total time: about 90 seconds.

---

## 1. Channel icon for `#mining-guardian` (15 sec)

1. Open Slack, click `#mining-guardian` in the left sidebar
2. Click the channel name at the top of the channel (where it says "mining-guardian")
3. Click the **Settings** tab in the popup
4. Scroll to **Channel icon** (or click the current default icon)
5. Click **Edit** or **Upload icon**
6. Pick `mining_guardian_mg_icon.png` from `~/Documents/GitHub/Mining Gaurdian/branding/`
7. Save

## 2. Channel icon for `#mining-guardian-alerts` (15 sec)

Same exact steps as #1 but for the alerts channel.

## 3. Mining Guardian bot avatar (30 sec)

1. Open https://api.slack.com/apps in your browser
2. Click on your **Mining Guardian** app in the list
3. In the left sidebar, click **Basic Information**
4. Scroll down to **Display Information** section
5. Click **+ Add app icon**
6. Upload `mining_guardian_mg_icon.png`
7. Save Changes
8. Optional: also fill in "Short description" with something like "Bitcoin mining fleet AI for BiXBiT USA"

The new bot avatar will appear next to every Mining Guardian message in `#mining-guardian` and `#mining-guardian-alerts` going forward (including in the demo replay if you ever scroll back).

## 4. (Optional) Workspace-wide custom emoji `:mining-guardian:` (30 sec)

So you can react to messages with the MG icon as an emoji.

1. In Slack, click the workspace name at the top-left
2. Tools & settings → Customize workspace
3. Click **Add Custom Emoji**
4. Name: `mining-guardian` (creates `:mining-guardian:`)
5. Upload `mining_guardian_mg_icon.png`
6. Save

Now you can react to any message with `:mining-guardian:` and it shows the MG badge.

---

That's it. You don't need to do any of these — they're polish, not function. But if you want the brand to feel locked-in across the workspace, the 90 seconds is worth it.

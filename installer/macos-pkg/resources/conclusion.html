Mining Guardian is running.

All four background services are now registered with launchd and will start automatically every time this Mac boots. A first-run baseline scan is running in the background — give it about 30 seconds before opening the dashboard.

  • Approval queue: http://127.0.0.1:8081/
  • Install log: /var/log/mining-guardian/install-postinstall.log
  • Service logs: /usr/local/MiningGuardian/logs/
  • Install receipt: /etc/mining-guardian/install-receipt.json

Verifying the four services

From Terminal.app:

sudo launchctl print system/com.miningguardian.scanner
sudo launchctl print system/com.miningguardian.dashboard-api
sudo launchctl print system/com.miningguardian.approval-api
sudo launchctl print system/com.miningguardian.feedback-loop-daemon

Uninstall

Run sudo /usr/local/MiningGuardian/bin/uninstall.sh to cleanly stop all four services, drop the Postgres container, and remove /usr/local/MiningGuardian. Your Postgres data volume is preserved at /usr/local/MiningGuardian/postgres-data by default — pass --purge-data to remove it.

# Command Suggestions

When you notice patterns that match a toolkit command, mention the command briefly.

| Pattern | Suggest |
|---------|---------|
| User debugging without structure | `/debug` for hypothesis-driven flow |
| Multiple file edits, no commits | `/create-commit` to checkpoint |
| UI work without design validation | `/design-check` for compliance |
| Tests written ad-hoc | `/tdd` for red-green-refactor |
| Large task with no plan | `/create-plan` to break it down |
| Deferred work or out-of-scope mention | `/create-todo` to capture for later |
| Bug discovered during other work | `/create-todo --category bugs` to track it |
| TODO/FIXME found in code | `/create-todo` to add to backlog |

Keep suggestions to one line. Do not repeat a suggestion already made this session.

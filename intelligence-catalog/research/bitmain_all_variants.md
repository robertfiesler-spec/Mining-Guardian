# Bitmain Antminer SHA-256 Bitcoin Miner — Complete Variant List

> **Coverage:** Every distinct SHA-256 Antminer variant ever produced or announced by Bitmain, from the S1 (2013) through the S23 series (2025–2026).  
> **Sources:** [Bitmain Support](https://support.bitmain.com), [ASIC Miner Value](https://www.asicminervalue.com/manufacturers/bitmain), [D-Central Technologies](https://d-central.tech), [Hashrate Index](https://hashrateindex.com/blog/antminers19-profitability-price/), [Forbes S23 announcement](https://www.forbes.com/sites/colinharper/2025/05/31/bitmain-reveals-the-antminer-s23-the-most-powerful-bitcoin-miner-ever/), Zeus Mining chip reference table.  
> **Notes:**  
> - Hashrate/power figures are **stock/typical** values at 25°C unless noted (hydro units: at 35°C inlet).  
> - Process nodes for S19-era chips: 7 nm (BM1398/BM1362/BM1360) and 5 nm (BM1366); S21-era: 5 nm (BM1368) and advanced 5 nm / 3 nm (BM1370); S9-era: 16 nm (BM1387); S7 and earlier: 28 nm (BM1385), 28 nm (BM1384/1382), 55 nm (BM1380).  
> - "Hashboards" count: 3 for almost all models; 4 for S1–S5 era and S15.  
> - Status: **Current** = still in production/sold new; **Discontinued** = no longer manufactured new.

## Notes on Naming Conventions

- **S19j** — "j" denotes a bin-selected cost-reduction variant of the S19 using a different chip (BM1362 vs BM1398)
- **S19j Pro** — "j Pro" = BM1362 chips in Pro hashboard configuration  
- **S19j Pro+** — incremental efficiency improvement, slightly different PCB design  
- **S19j Pro++** — further step-up in the j-Pro+ line  
- **S19k Pro** — "k" = cost-reduction S19 variant using 5nm BM1366 in a lower-power configuration  
- **S19 XP** — "XP" = eXtra Performance; first 5nm Antminer air-cooled  
- **S19j XP** — j-bin version of the XP, 151 TH/s at 21.5 J/TH  
- **S19 Pro++** — 2024 refresh using BM1366 chips in the older S19 chassis  
- **S19 Pro-A** — "A" suffix = regional/batch variant, 2024 release  
- **S19al** — "al" = alternative/low-cost binning of S19j, uses BM1362AJ  
- **S19i** — "i" = improved; uses BM1360BB chip, intermediate model  
- **S19a / S19a Pro** — "a" = specific batch variant using BM1398AC/AD chips  
- **S21 Hyd vs S21+ Hyd** — S21+ Hyd is a revised hydro variant; both 335 TH/s but S21+ Hyd 319 is the Feb 2025 update at 15 J/TH  
- **S21e XP Hyd** — "e" = enterprise; single-unit 430 TH/s hydro variant  
- **S21e XP Hyd 3U** — rack-scale 3U form factor, 860 TH/s (essentially 2× S21e XP Hyd in one chassis)  
- **T21** — T-series = budget/value tier of S21; uses same BM1368 chip in two firmware modes: NEM (Normal Energy Mode: 190 TH/s, 3,610W, 19 J/TH) and HEM (High Energy Mode: 233 TH/s, 5,126W, 22 J/TH). Both modes listed as separate rows per Bitmain official specs  
- **S23 IMM** — purpose-built immersion chassis (no fans, no water block connectors — designed for dielectric fluid baths)
